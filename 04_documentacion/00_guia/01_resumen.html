Introducción

El presente documento contiene enlaces y referencias para los conceptos utilizados en la construcción de la carpeta de data que se encuentra en la siguiente ruta del Disco B "B:- Ministerio de Educación_B\0. Data".

  1. Data en formato tidy:

Datos en formato tidy proporcionan una forma estandarizada para trabajar con datos, puedes leer mas en este articulo de Towards Data Science: https://towardsdatascience.com/what-is-tidy-data-d58bb9ad2458

En general, la data en formato tidy tiene que cumplir con las siguientes reglas:

  1. Cada variable es una columna

  2. Cada observacion es una fila

Usaremos el formato tidy para convertir las tablas de datos “raw” o crudas en tablas de datos clean.
